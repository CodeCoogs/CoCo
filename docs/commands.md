# Commands

Prefix `coco-`

## The `help` command
_Access: everyone_

__No arguments__
> coco-help

Displays all commands

__info__
> coco-help info

Display info command

__stonks__
> coco-help stonks

Display stonks command


## The `info` command
_Access: wizard_

__No arguments__
> coco-info

Does nothing

__rules__
> coco-info rules

Sends/Edits message in #rules

__officers__
> coco-info officers

Sends/Edits message in #officers

__socials__
> coco-info socials

Sends/Edits message in #socials
